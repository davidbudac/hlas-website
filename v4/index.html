01
Local AI
On-device transcription powered by WhisperKit and OpenAI Whisper models. Your audio never leaves your Mac. Zero latency from network round-trips. Works offline, works on planes, works in classified environments.
02
Remote GPU
For maximum accuracy or longer recordings, route audio to a remote GPU server running larger Whisper models. Ideal for older hardware or when you need the absolute best transcription quality.
03
Auto-Detect
Supports 60+ languages with automatic language detection. Start speaking in English, switch to Slovak, continue in German. Hlas follows along without any manual language switching.
04
Translation
Speak in one language, get text in another. Dual translation via Apple Translation Framework and Whisper's built-in translation. Dictate in Slovak, paste in English. Seamless.
05
Smart Text
Automatic smart punctuation, filler word removal, and a personal dictionary that learns your terminology. Technical terms, names, acronyms — taught once, recognized forever.
06
Universal
Works with every app on macOS. Browsers, code editors, design tools, terminals, messaging apps, document editors. If it accepts text input, Hlas works with it. No plugins, no integrations, no setup.
